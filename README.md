# BPMS - Business Process Management System (WinForms)

A clean and modular Windows Forms application built with .NET Framework, demonstrating:

## ✨ Features
- SQL Server data access layer
- Input validation system
- Layered architecture (Core / Infrastructure / UI)

## 🏗 Architecture
- **Core** → Business logic & services
- **Infrastructure** → Database & file handling
- **WinForms** → UI layer

## 🧠 Key Concepts
- Separation of Concerns
- Reusable Data Access Layer
- Interface-based design
- Clean code principles

## 🛠 Tech Stack
- C#
- .NET Framework
- SQL Server
- WinForms

## 📌 Notes
This project is structured to reflect production-level coding standards suitable for enterprise environments.
